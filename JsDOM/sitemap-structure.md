# SEO-Optimized Page Hierarchy & Keyword Targeting

## Page Structure (Max 3 Levels Deep)

### Level 1: Homepage
- **URL**: `/`
- **Target Keyword**: "Mobile repair and sales [Your Main City]"
- **Purpose**: Brand homepage with overview of all services

### Level 2: Main Category Pages

#### Mobile Sales Category
- **URL**: `/mobile-sales/`
- **Target Keyword**: "Mobile sales"
- **Purpose**: Landing page for all mobile sales services

#### Mobile Repair Category  
- **URL**: `/mobile-repair/`
- **Target Keyword**: "Mobile repair"
- **Purpose**: Landing page for all repair services

#### Locations Category
- **URL**: `/locations/`
- **Target Keyword**: "Mobile service locations"
- **Purpose**: Overview of all branch locations

## SEO Benefits of This Structure

1. **Shallow Depth**: Maximum 3 levels prevents deep crawling issues
2. **Clear Hierarchy**: Logical flow from general to specific
3. **Keyword Focus**: Each page targets one primary keyword
4. **Internal Linking**: Easy navigation between related services
5. **Local SEO**: Location pages target city-specific searches
6. **Scalability**: Easy to add new services or locations
